# Interust
해당 라이브러리는 Rust 문법을 사용하는 인터프리터 엔진을 제공합니다.
### v0.1.0
사용 가능한 타입이 제한되어 있습니다.
쉽게 원시 타입을 추가할 수 있도록 각 타입의 바이트 코드 자리는 비워두었습니다.
#### 사용 가능한 변수 타입
- i64 : 64bit 부호 있는 정수 
- f64 : 64bit 부호 있는 실수
- bool : 1bit 논리 값
- string : 최대 길이가 usize의 최대값인 문자열
